Linux

O que é?

Linux foi criado por um universitário chamado Linus Torvalds lá em 1991,
o cara não criou um dos projetos mais importantes da informática por querer,
ele só queria algo legal que ajudaria as pessoas para estudar, dentro de casa mesmo e tal.

Ao contrário do que todo mundo acha, Linux não é um sistema operacional
como os outros em si, mas um open source, um núcleo em que vários
programadores podem usar para criar sistemas operacionais. Por isso
existem vários sistemas operacionais diferentes em que “Linux”
está na frente de seu nome (popularmente), como Linux Mint,
Linux Ubuntu, Linux Debian e etc.

Por que usar?

O Linux não é tão usado por todos os usuários do mundo inteiro
como o Windows, mas ainda é bem popular, principalmente entre
programadores e pessoas que precisam administrar servidores em
Redes de Computadores, por haver maior facilidade em comparação
com Windows e Mac quando usar o prompt de comando, por exemplo.

Apesar disso, o Linux também é conhecido por sua baixa latência,
ou seja, é bem leve, roda em qualquer tipo de máquina. Além de
a maioria das distribuições serem gratuitas

Como já dito, ele é bastante usado por programadores, isso é porque
o próprio sistema do Linux já vem com interpretadores e compiladores
de algumas linguagens de programação, como python, shell e gcc.
Além de algumas extensões que já possuem um ambiente de desenvolvimento nativo.

Outra vantagem é o código do Linux ser aberto, ou seja, se você quiser fazer
sua própria extensão Linux, seu próprio SO.

Como Usar

Normalmente, ao comprar um computador novinho,
a maioria já vem com um sistema operacional
instalado, geralmente Windows.

Para instalar qualquer SO Linux só basta acessar
o site oficial de cada extensão e baixar a versão
que mais te atende, mas existem as mais populares
entre todas que existem.

Como já dito, o Linux possui várias distribuições,
aqui vamos citar as mais usadas e ensinar como instalá-las.

Ubuntu

O Ubuntu é uma das distribuições Linux mais famosas.
Todo esse sucesso tem seus motivos: o sistema é fácil de
usar e agrada usuários iniciantes no mundo do software livre.
Além disso, ele é uma boa opção para quem tem uma máquina com
problemas de desempenho, por ser mais leve que seus concorrentes

Alguns requisitos são necessários para a instalação do sistema operacional:

  • 25 GB de espaço de armazenamento livre;
  • 4 GB de memória RAM;
  • Processador dual-core de, no mínimo, 2 GHz;
  • Uma porta USB;
  • Um pendrive de, no mínimo, 8 GB (se tiver 12 ou mais é melhor).

Faça o download do Ubuntu em https://ubuntu.com/download/desktop

Mint

O Linux Mint é muito parecido com o Ubuntu e o Debian,
isso é porque ele foi baseado neles, E ele possui
uma interface muito intuitiva, como a do Ubuntu,
mas ele possui suas próprias ferramentas,
e também é gratuito.

Requisitos de sistema:

  • 2 GB de RAM (4 GB recomendados para um uso confortável).
  • 20 GB de espaço em disco (100 GB recomendados).
  • Resolução de 1024 × 768 (em resoluções mais baixas, pressione ALT
    para arrastar as janelas com o mouse se elas não couberem na tela).

Faça o download do Mint em https://www.linuxmint.com/download.php

Deepin

O Linux Deepin é totalmente baseado em Debian, o
seu diferencial é a área de trabalho personalizada,
chamada DD, que é como se fosse um kit de ferramentas.

Requisitos de sistema:

  • Mais de 2 GB de RAM (recomenda-se 4 GB)
  • Intel Pentium IV 2GHz (ou superior)
  • 25 GB de espaço livre em disco (ou superior)
  • Unidade USB inicializável.
  • Conexão de Internet estável.

Faça o download do Deepin em https://www.deepin.org/index/en

Comandos Básicos
para o prompt de comando:

O que é o prompt de comando?

O prompt de comando é um programa que emula
um campo de entrada em uma tela de interface
do usuário, onde você pode desde mover
arquivos à executar funções
administrativas no computador.



Para abrir o prompt de commando no linux: Ctrl+Alt+T



Alguns comandos simples e para que servem:

Para saber qualquer comando:

  • help: esse comando te mostra todos os comandos de que você precisa



Continuando, comandos para manutenção e monitoramento mais usados:

  • ls: Lista todos os arquivos do diretório
  • df: Mostra a quantidade de espaço usada no disco rígido
  • top: Mostra o uso da memória
  • cd: Acessa uma determinada pasta (diretório)
  • mkdir: Cria um diretório
  • rm: Remove um arquivo/diretório
  • cat: Abre um arquivo
  • vi: Abre o editor vi para editar/criar arquivos



Comandos de notícias ou rede:

  • netstat: Mostra o estado da rede
  • rsh: Um shell em outros sistemas UNIX
  • ssh: Versão segura do rsh
  • nmap: Poderoso port-scan, para visualizarmos portas abertas num dado host
  • ifconfig: Visualizar os ips da nossa máquina, entre outras funções relacionadas
    com ips
  • ping: Pingar um determinado host, ou seja, enviar pacotes icmp para um
    determinado host e medir tempos de resposta



Comandos para digestão de arquivos e diretórios

  • chown: Mudar o dono ou grupo de um arquivo ou diretório
  • chgrp: Mudar o grupo de um arquivo ou diretório
  • cmp: Compara dois arquivos
  • comm: Selecionar ou rejeitar linhas comuns a dois arquivos selecionados
  • cp: Copia arquivos, como o copy do MS-DOS
  • crypt: Encripta ou Descripta arquivos (apenas CCWF)
  • diff: Compara o conteúdo de dois arquivos ASCII
  • file: Determina o tipo de arquivo
  • grep: Procura um arquivo por um padrão
  • gzip: Comprime ou expande arquivo
  • ln: Cria um link a um arquivo
  • lsof: Lista os arquivos abertos, vem de list open files
  • mv: Move ou renomeia arquivos ou diretórios
  • pwd: Mostra-nos o caminho por inteiro do diretório em que nos encontramos em dado momento
  • quota: Mostra-nos o uso do disco e os limites
  • rmdir: Apaga diretório, vem de remove directory
  • stat: Mostra o estado de um arquivo
  • sync: Faz um flush aos buffers do sistema de arquivos, sincroniza os dados no disco com a memória
  • sort: Ordena, une ou compara texto
  • tar: Cria ou extrai arquivos
  • tee: Copia o input para um standard output e outros arquivos
  • tr: Traduz caracteres
  • umask: Muda as proteções de arquivos
  • uncompress: Restaura um arquivo comprimido
  • uniq: Reporta ou apaga linhas repetidas num arquivo
  • wc: Conta linhas, palavras e mesmo caracteres num arquivo
